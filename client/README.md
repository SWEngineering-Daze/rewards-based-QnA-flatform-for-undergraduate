# 클라이언트

## Environment Requirements

- Node `16`
- NPM `8`

## Installation

### Local (Development)

1. install dependencies

```
npm install
```

2. launch development server

```
npm run dev
```

### Production (Server)

1. install dependencies

```
npm install
```

2. build

```
npm run build
```

3. launch production server (with pm2)

```
pm2 start ecosystem.config.js
```
